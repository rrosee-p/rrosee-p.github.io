Code Log 1: HTML Semantic Tags

What is the difference between non-semantic and semantic tags in HTML?

Semantics is the use of HTML tags that accurately reflect the content they contain. Clearly defines its content.

Non-semnatic tell nothing about their content. They are used purely to define the presentation or layout of a page.

What are two (or more) benefits of using semantic tags?

Some benefits include:

  • Easier to read
  • Greater accessibility
  • Consistent Code
