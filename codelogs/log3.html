Images

Images can be added to websites using HTML or CSS. Here's how they work and what makes them different:

Using <img> in HTML

You use the <img> tag to show an image directly on the page.

<img src="image.jpg" alt="Description of image">

Using background-image in CSS

You can also set an image as the background of a box or section using CSS.

.background-box {
  background-image: url('image.jpg');
  background-size: cover;
  background-position: center;
}
Background Image Example

What’s the Difference?

  • <img>: Used for important images (like photos, logos).
  • background-image: Used for decoration (like a banner or section background).
  • <img> can be seen by screen readers. background-image cannot.
  • <img> is part of the page content. background-image is only style.

Other Useful CSS for Background Images

  • background-size: Makes the image fit. Use cover or contain.
  • background-position: Moves the image (like center, top right).
  • background-repeat: Lets the image repeat or not (no-repeat).

Dimensions vs Resolution

Dimensions = width and height in pixels. Resolution = how clear the image looks, especially when printed.

Extra Credit: Image Formats

  • GIF: Small files, supports animation. Good for simple graphics.
  • PNG: Clear images with transparency. Great for logos and icons.
  • JPG: Small size, great for photos. Doesn’t support transparency.
  • WebP: Newer format, small and high quality. Good for web use.
  • SVG: Vector format. Stays sharp at any size. Best for icons and logos.
