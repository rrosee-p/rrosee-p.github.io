Images

Images are an essential part of web design and can be included in web pages using both HTML and CSS. Below, we explore the differences between these methods, important CSS properties, and general image concepts.

Using Images in HTML

You can use the <img> tag to embed images directly into your HTML content:

<img src="image.jpg" alt="A description of the image">

Using Background Images in CSS

You can apply background images to HTML elements using CSS, like this:

background-image: url('image.jpg');
background-size: cover;
background-position: center;

Other Useful CSS Properties for Background Images

  • background-size: Controls how the background image is scaled. Common values: cover, contain.
  • background-position: Determines where the background image is placed inside the element. E.g., center, top right.
  • background-repeat: Controls whether the image repeats. Common values: no-repeat, repeat-x, repeat-y.

Bitmap Image Dimensions vs. Resolution

Dimensions refer to the width and height of an image in pixels, while resolution refers to the pixel density (e.g., DPI), affecting print quality but not display size on screens.
