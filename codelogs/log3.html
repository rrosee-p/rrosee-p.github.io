How can images be used in HTML and CSS?

Images can be added to a webpage using HTML or CSS. Here's how they work and how to use them.

1. Adding Images with HTML

The <img> tag is used to show an image on the page. Example:

<img src="image.jpg" alt="Description of image">

2. Adding Images with CSS

You can also add an image as a background using CSS. Example:

.background-box {
  background-image: url('image.jpg');
  background-size: cover;
  background-position: center;
}
Background Image Example

What's the Difference?

  • <img>: Adds an image as part of the page content.
  • background-image: Adds an image as a decoration (background).
  • <img> is good for photos, product images, etc.
  • background-image is good for banners, patterns, and design.

Other CSS Properties for Background Images

  • background-size: Controls how the image fits the space (cover, contain).
  • background-position: Sets where the image is placed (like center or top right).
  • background-repeat: Tells the image to repeat or not (no-repeat, repeat-x).

Dimensions vs Resolution

Dimensions are the width and height of an image in pixels. Resolution is how detailed the image is, usually shown in DPI (dots per inch).

Image File Formats

  • GIF: Supports animation, limited colors. Good for simple animated icons.
  • PNG: Supports transparency, no quality loss. Great for logos and clear graphics.
  • JPG/JPEG: Smaller file size with some quality loss. Best for photos.
  • WebP: New format with small size and good quality. Great for modern websites.
  • SVG: Vector format. Sharp at any size. Best for icons, logos, and simple shapes.
