CSS Color

What do these CSS properties do?

Color

  • Sets the text color (foreground color) of an element.

    • Applies to:

  • Text inside the element (including inline text, headings, paragraphs, links, etc.)

    • Background-Color

  • Sets the background color of an element.

    • Applies to:

  • The entire background area of the element, including padding (but not margin).

    • Border-Color

  • Sets the color of the element's border.
  • A border must be defined using the proper functions, or it won't appear. Setting border-color alone does nothing unless a border is visible.
